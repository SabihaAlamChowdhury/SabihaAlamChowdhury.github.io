iii. Technical Work

Research Projects

🧭
Indoor Scene Detection & Navigation System
A flagship multimodal pipeline combining image and text streams for deep indoor scene understanding. Employs Neurosymbolic AI — fusing neural perception with symbolic spatial reasoning via Answer Set Programming — to enable assistive navigation for visually impaired users. Bridges the gap between raw visual data and structured, explainable spatial knowledge.
Neurosymbolic AI YOLO VQA ASP / Clingo Multimodal ML
🚚
DeepTransit: Urban Delivery Simulation
Collaborated on a conveyor network simulation to resolve urban logistics bottlenecks. Applied deep learning models for optimised multi-agent package routing.
Deep Learning Discrete Event Sim Multi-Agent

8086 String Manipulation System
Low-level system for complex string manipulation with hardware-software interfacing via 8086 Assembly. Demonstrates deep knowledge of memory management and interrupt I/O.
